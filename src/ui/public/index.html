    Bluesky Settings

    Fetches your Bluesky timeline using the official AT Protocol API

    How to get an App Password
    1. Go to bsky.app/settings/app-passwords
    2. Click "Add App Password"
    3. Give it a name (e.g., "Slowfeed")
    4. Copy the generated password (format: xxxx-xxxx-xxxx-xxxx)
    5. Paste it here

    Note: App passwords are separate from your main password and can be revoked anytime. Never share your main Bluesky password.

    YouTube Settings

    Scrapes youtube.com/feed/subscriptions using your browser cookies

    How to get long-lasting YouTube cookies

    Use an incognito/private window to get cookies that last months instead of days:

    1. Install a cookie export extension like Get cookies.txt LOCALLY (Chrome) or cookies.txt (Firefox)
    2. Open a new incognito/private browsing window
    3. Go to accounts.google.com and log in
    4. Navigate to youtube.com/robots.txt
    5. Click the cookie export extension and export cookies for the current site
    6. Immediately close the incognito window (do not browse further)
    7. Paste the exported cookies.txt content here

    Why incognito? Cookies from regular sessions are tied to your browsing activity and expire quickly. Incognito cookies created for a minimal session last much longer (often months).

    Format: Both Netscape cookies.txt format (from extensions) and raw Cookie header format (from DevTools) are supported.

    Reddit Settings

    Scrapes old.reddit.com (optionally with your cookies for personalized feed)

    How to get Reddit cookies

    Without cookies, Slowfeed will scrape the public Reddit front page. With cookies, you get your personalized home feed based on your subscriptions.

    1. Open old.reddit.com in Chrome/Firefox and make sure you're logged in
    2. Press F12 to open Developer Tools
    3. Go to the Network tab
    4. Refresh the page (F5)
    5. Click on the first request to "old.reddit.com" in the list
    6. In the right panel, scroll down to Request Headers
    7. Find the Cookie header and copy its entire value
    8. Paste the value here

    Important: Use old.reddit.com, not www.reddit.com. The cookie format is different.

    Discord Settings

    Fetches messages from selected Discord channels using your user token

    How to get your Discord token
    1. Open discord.com/app in Chrome/Firefox (not the desktop app)
    2. Press F12 to open Developer Tools
    3. Go to the Network tab
    4. In the Filter box, type api to filter requests
    5. Send a message in any channel, or switch channels
    6. Click on any request to discord.com/api in the list
    7. In the right panel, scroll down to Request Headers
    8. Find the Authorization header and copy its value (do NOT include the word "Authorization:")
    9. Paste the value here

    Security note: Your Discord token grants full access to your account. Keep it secret and never share it. Slowfeed stores it locally and only uses it to read messages from channels you select.
